# WhatsApp Stat

A zero-backend tool to parse and visualize WhatsApp chat exports in your browser.

## Features

- Parses common WhatsApp exported `.txt` formats (Android and iOS style timestamps)
- Handles multiline messages
- Shows:
  - message count, participant count, word count, active days
  - top participants
  - activity by hour and weekday
  - monthly timeline
  - top words
- All processing stays local in the browser

## Run

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite (usually `http://localhost:5173`).
